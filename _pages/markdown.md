## 关于本站

这个网站是我的个人主页，基于 [Academic Pages](https://github.com/academicpages/academicpages.github.io) 模板搭建，托管在 **GitHub Pages** 上。

### 技术栈

- **静态站点生成器：** [Jekyll](https://jekyllrb.com/)
- **主题：** [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/)
- **托管：** [GitHub Pages](https://pages.github.com)（免费、自动部署）

### 工作流程

1. 在本地编辑 Markdown 文件
2. `git push` 推送到 [GitHub 仓库](https://github.com/awen11123/awen11123.github.io)
3. GitHub Pages 自动构建并部署
4. 几分钟后网站即更新
